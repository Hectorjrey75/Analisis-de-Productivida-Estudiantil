## 📋 Descripción
Análisis de cómo los hábitos digitales modernos afectan el rendimiento académico y la productividad estudiantil.

## 🎯 Objetivos
- Analizar correlación entre hábitos digitales y rendimiento académico
- Desarrollar modelos predictivos de productividad estudiantil
- Identificar factores clave de éxito académico
- Proveer recomendaciones basadas en datos
